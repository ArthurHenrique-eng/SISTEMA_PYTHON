#SISTEMA_PYTHON

Sistema de autenticação desktop desenvolvido em Python puro, sem dependências externas.
Conta com interface gráfica construída em tkinter, módulos separados por responsabilidade
(validação, usuário, navegador, configuração) e limite de tentativas de login.
Ao autenticar com sucesso, redireciona para uma URL configurável.

##Como rodar

```bash
python main.py
